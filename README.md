# Verdaccio

Verdaccio is an open-source, lightweight private npm proxy registry built in Node.js. It enables organizations to host private packages, cache upstream registry packages (e.g., npmjs.org), and chain multiple registries through a single endpoint — all without a full database setup or commercial licensing fees.
